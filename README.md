# Priya Rauniyar — Portfolio

A modern, premium portfolio website built with **React + Vite + Tailwind CSS + Framer Motion**.

## ✨ Features

- 🌙 Dark theme with glassmorphism effects
- 🎯 Custom animated cursor
- ✨ Canvas particle background (stars + floating orbs)
- 🎬 Framer Motion animations throughout
- 📱 Fully responsive (mobile + desktop)
- ⌨️ Typewriter animation in Hero
- 📊 Animated skill progress bars
- 🎴 Interactive project cards with hover glow
- 🔝 Scroll to top button
- ⚡ Loading animation
- 🧭 Active section navbar highlight

## 🛠️ Tech Stack

- **React 18** + **Vite**
- **Tailwind CSS** (utility-first styling)
- **Framer Motion** (animations)
- **React Scroll** (smooth nav scrolling)
- **React Type Animation** (typewriter effect)
- **React Icons** (icon library)

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
